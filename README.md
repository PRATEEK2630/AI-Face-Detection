# 👤 Face Detection using OpenCV

This project is a simple real-time face detection system built using **Python** and **OpenCV**. It uses a pre-trained Haar Cascade classifier to detect human faces through a webcam.

---

## 🚀 Features

- Real-time face detection using webcam  
- Uses Haar Cascade Classifier  
- Draws bounding boxes around detected faces  
- Lightweight and beginner-friendly  

---

## 🛠️ Technologies Used

- Python  
- OpenCV (cv2)  
- Haar Cascade Classifier
